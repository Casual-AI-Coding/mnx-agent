## 四、技术决策记录

### Loop 节点实现方案
采用 subNodes/subEdges 模式，支持：
- items 数组迭代
- maxIterations 限制
- condition 条件终止
- 循环上下文变量 (item, index)

### Condition 分支实现方案
使用 sourceHandle 标识分支：
- 'true' → 条件为真时执行
- 'false' → 条件为假时执行
- 无 sourceHandle → 向后兼容，始终执行

### Webhook 恢复方案
- 表结构: 与归档版本一致
- 投递策略: 异步队列 + 重试 + HMAC签名验证
- 事件类型: on_start, on_success, on_failure
