# mnx-agent v2.0 Roadmap

> 版本规划文档 - 记录 v2.0 版本的功能目标和发展方向

## 概述

v2.0 版本将专注于完善现有功能、增强用户体验、提升系统稳定性，并补充生产环境必需的能力。

---

## Roadmap 项目列表

### P0 - 必须修复（阻塞发布）

| ID | 项目 | 说明 | 状态 |
|----|------|------|------|
| R-013 | Workflow Marketplace 修复 | 创建缺失的 `src/data/workflow-templates.ts`，修复页面空白问题 | 待办 |

---

### P1 - 核心功能完善（基础能力）

| ID | 项目 | 说明 | 状态 |
|----|------|------|------|
| R-003 | 云端备份 | media 数据云端备份，扩展本地快照脚本到 Backblaze B2/Cloudflare R2 | 待办 |
| R-006 | 工作流/DLQ 投产验证 | 验证工作流引擎、定时任务、死信队列实际可用，编写使用文档 | 待办 |
| R-004 | 审计日志/媒体记录字段补充 | 添加更多上下文信息（如 API 响应时间、完整请求参数等） | 待办 |

---

### P2 - 用户体验增强（关键功能）

| ID | 项目 | 说明 | 状态 |
|----|------|------|------|
| R-005 | 应用主页 Dashboard | 概览页面，展示 API 使用量、配额状态、最近任务、快速操作入口 | 待办 |
| R-009 | 请求历史回放 | 保存历史请求完整参数，一键重新执行相同请求 | 待办 |

| R-011 | 帮助中心 | 内置 FAQ、API 文档链接、常见错误解决方案 | 待办 |

---

### P3 - 系统管理（管理能力）

| R-002 | 资源管理完善 | 素材管理增强、素材使用统计、模板管理重构 | 待办 |

---

### P4 - 技术优化（长期演进）

| ID | 项目 | 说明 | 状态 |
|----|------|------|------|

## 版本里程碑

### v2.0-alpha
- 完成 P0（Workflow Marketplace 修复）
- 完成 P1（云端备份、投产验证）

### v2.0-beta
- 完成 P2（Dashboard、请求回放、新手引导、帮助中心）

### v2.0-release
- 完成 P3（系统管理）
- 完成 P4 部分优化项目
