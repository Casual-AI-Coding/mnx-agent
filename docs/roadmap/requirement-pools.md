## ID 编号规则

> 需求 ID 采用 `R-XXX` 格式，数字连续递增。

**编号原则**:
- 新需求使用当前最大 ID + 1
- 删除的需求 ID 直接移除，不保留空号
- ID 连续编号，便于追踪和引用

**当前 ID 范围**: R-001 ~ R-024（共 24 个需求）
